# 🌈 Sceptiques du Québec

Site officiel : https://sceptiquesduquebec.com/  
Repo : https://github.com/Sceptiques-du-Quebec/sceptiques-du-quebec.github.io

---

## 🚀 Installation

Clone le projet :

```bash
git clone https://github.com/Sceptiques-du-Quebec/sceptiques-du-quebec.github.io.git
cd sceptiques-du-quebec.github.io
npm install
code .
````

---

## 🛠️ Commandes

### Développement (watch)

Compile automatiquement lors des changements :

```bash
npm run watch
```

---

### Build

Compile le projet :

```bash
npm run build
```

---

### Export

Génère la version statique du site :

```bash
npm run export
```

---

## 🧠 Objectif

Sceptiques du Québec vise à promouvoir la pensée critique et à dénoncer :

* la désinformation
* l'intolérance
* les discours pseudoscientifiques
